# How to run the Virtual Try-On site

1. Install dependencies:
// turbo
```bash
npm install
```

2. Start the development server:
// turbo
```bash
npm run dev
```

3. Open the URL provided (usually http://localhost:5173).
4. Enter your Segmind API key in the UI.
5. Upload images and generate your try-on!
